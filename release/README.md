## How to Install

1. Download the APK file from above.
2. On your Android phone, go to **Settings → Security → Enable "Install from Unknown Sources"**.
3. Open the downloaded APK and tap **Install**.
4. Launch **Nyaya-Uday** from your app drawer.

> **Minimum requirement:** Android 5.0 (Lollipop) or higher.
